# Installation
#### Clone:
```sh
git clone https://github.com/Broken-Lab/script -b master && cd script
```
#### Permission:
```sh
chmod +x *
```
#### Run Script:
```sh
bash <script_name>.sh
```
#### Direct method:
```sh
bash <(curl -sL https://raw.githubusercontent.com/Broken-Lab/script/master/<script_name>.sh)
```
